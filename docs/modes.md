# Modes

The system supports several working modes.

## DMAIC mode
Use the full Define, Measure, Analyze, Improve, Control structure.

## Kaizen mode
Use a lighter-weight continuous improvement framing for smaller improvement loops.

## Root cause mode
Focus on likely causes, evidence gaps, and bottleneck signals.

## Process waste mode
Focus on likely waste patterns such as waiting, rework, excess handoffs, or unclear ownership.

## Control plan mode
Focus on metrics, review cadence, owners, and threshold-based triggers.
