# Sample Output Overview

Running the demo with:

```bash
python run_demo.py --input examples/sample_project.json --mode dmaic --audience pm
```

produces a structured markdown summary with:
- cleaned-up problem statement
- likely CTQs
- SIPOC draft
- DMAIC structure
- possible root causes
- suggested metrics
- suggested improvement actions
- control plan draft
- action tracker
- project memory
- role-aware summary

The output is intentionally structured and evidence-tagged rather than written as generic advice.
